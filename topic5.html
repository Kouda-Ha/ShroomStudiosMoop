Topic 5: Who? What? Where? When? How?

To use the question words, you need to know about some sentence structure like in Topic 3.

A few pointers:

  • Adding か to the end of a sentence turns it into a question. For example:
    ケーキです = It's cake.
    ケーキですか = Is it cake?
  • Remember でした is past positive, with か on the end you're asking if there was something. For example:
    ケーキでしたか = Was there cake?
  • Adding a question word, to ask a basic question. For example:
    だれですか = Who is it?
    いつですか = When is it?
    なんですか = What is it?
    Again, using past positive would ask 'Who was it?', 'When was it?', 'What was it?' and etc.
  • It also works with negatives both past or present. For example:
    ケーキをたべませんでしたか is asking 'You have never eaten cake?', and etc.
